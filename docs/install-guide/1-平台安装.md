# 平台安装

## 下载repo源

集群每个节点都需要操作

```sh
# 下载安装repo源,集群每个节点都需要操作
wget https://redoop-mirrors.oss-cn-hangzhou.aliyuncs.com/apache/snapshots/ambari/2.8.0/centos/7/x86_64/ambari.repo -O /etc/yum.repos.d/ambari.repo

yum clean all
yum makecache
```

## 安装ambari-server

主节点安装即可

```sh
# 安装ambari-server
yum install ambari-server -y

# 导入元数据
mysql --database=ambari -e "source /var/lib/ambari-server/resource/Ambari-DDL-MYSQL-CREATE.sql" -uroot -p
```

## 初始化ambari-server

### 配置jdk位置

ambari-server setup

### 配置mysql-connector-java.jar

```sh
# 初始化ambari
ambari-server setup --jdbc-db=mysql --jdbc-dirver=usr/usr/share/java/mysql-connector-java.jar

# 如果没有mysql-connector-java.jar可下载
wget https://repo1.maven.org/maven2/mysql/mysql-connector-java/5.1.47/mysql-connector-java-5.1.47.jar -O /usr/share/java/mysql-connector-java.jar
```

## 启动并检查Ambari状态

```shell
#启动 Ambari 服务
ambari-server start

#检查启动状态
ambari-server status
```

至此，服务组件启动完毕
访问8080端口即可安装大数据组件
